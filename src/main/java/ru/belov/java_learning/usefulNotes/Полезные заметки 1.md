При локальном запуске приложения, если происходят запросы к внешним ресурсам (например стороннему api) может возникнуть ошибка по аналогии с kafka описанная в `Локальный запуск приложения.md` 
Ошибка будет примерно такой
`Caused by: feign.RetryableException: PKIX path building failed: sun.security.provider.certpath.SunCertPathBuilderException: unable to find valid certification path to requested target executing POST https://dh2.efir-net.ru/v2/Archive/EndOfDay`
- самый простой вариант будет отключить ssl валидацию для такого рода запросов, то есть настроить **Feign** -  HTTP-клиент для Java, который позволяет вызывать сторонние REST-API как обычные Java-методы, без ручной работы с `HttpClient`, `RestTemplate` и т.п.
- я перепробовал множество вариантов, рабочим оказался следующий
	- Если feign подтягивался неявно, то явно указать зависимость `implementation 'io.github.openfeign:feign-okhttp:11.10'`
	- в `application.yaml` указать 
		```
		feign:  
		  httpclient:  
		    enabled: false  
		    disable-ssl-validation: true  
		  okhttp:  
		    enabled: true
		```
